## 关键词检测函数

```javascript
/**
 * 检测内容是否为目标内容
 * @param {Object} postInfo - 帖子信息（包含文本、图片描述等）
 * @param {Array} targetKeywords - 目标关键词数组
 * @returns {boolean} 是否为目标内容
 */
function isTargetContent(postInfo, targetKeywords) {
  // 合并所有文本信息
  const allText = [
    postInfo.text || '',
    postInfo.authorName || '',
    postInfo.pageName || '',
    postInfo.groupName || '',
    ...(postInfo.hashtags || [])
  ].join(' ').toLowerCase();
  
  // 检查是否包含任一关键词
  return targetKeywords.some(keyword => 
    allText.includes(keyword.toLowerCase())
  );
}

/**
 * 从页面快照中提取帖子信息
 * @param {Object} snapshot - 页面快照
 * @returns {Object} 帖子信息
 */
function extractPostInfo(snapshot) {
  const allText = snapshot.innerText || '';
  
  // 提取标签（#开头的词）
  const hashtags = allText.match(/#\w+/g) || [];
  
  // 提取@提及
  const mentions = allText.match(/@\w+/g) || [];
  
  return {
    allText: allText.toLowerCase(),
    hashtags: hashtags.map(t => t.slice(1).toLowerCase()),
    mentions: mentions.map(m => m.slice(1).toLowerCase())
  };
}
```

## Facebook特定检测

Facebook内容结构与TikTok不同，需要额外检测：

```javascript
/**
 * 检测Facebook帖子类型
 * @param {Object} postElement - 帖子DOM元素
 * @returns {string} 帖子类型
 */
function detectPostType(postElement) {
  // 检测视频
  if (postElement.querySelector('video')) {
    return 'video';
  }
  
  // 检测Reel
  if (postElement.querySelector('[aria-label*="Reel"]') || 
      window.location.href.includes('/reel/')) {
    return 'reel';
  }
  
  // 检测图片
  if (postElement.querySelector('img[data-visualcompletion]')) {
    return 'image';
  }
  
  // 检测链接分享
  if (postElement.querySelector('[data-lynx-mode]')) {
    return 'link';
  }
  
  // 默认文字帖子
  return 'text';
}

/**
 * 检测是否为广告
 * @param {Object} postElement - 帖子DOM元素
 * @returns {boolean} 是否为广告
 */
function isAdPost(postElement) {
  const text = postElement.innerText || '';
  return text.includes('Sponsored') || 
         text.includes('赞助') ||
         postElement.querySelector('[aria-label*="Sponsored"]') !== null;
}
```

## 排除关键词

某些关键词可能导致误判，需要排除：

```javascript
const excludeKeywords = {
  // 通用排除
  common: [
    'sponsored', 'ad', 'advertisement',
    'casino', 'gambling', 'betting',
    'scam', 'fake', 'clickbait'
  ],
  
  // 健身类排除（避免把美妆误判为健身）
  fitness: [
    'makeup', 'cosmetics', 'lipstick',
    'fashion show', 'runway'
  ],
  
  // 电商类排除
  ecommerce: [
    'giveaway scam', 'too good to be true'
  ]
};

/**
 * 带排除检测的目标内容判断
 */
function isTargetContentWithExclusion(postInfo, targetKeywords, category = 'common') {
  const allText = postInfo.allText;
  
  // 先检查排除词
  const excludeList = [
    ...excludeKeywords.common,
    ...(excludeKeywords[category] || [])
  ];
  
  if (excludeList.some(kw => allText.includes(kw.toLowerCase()))) {
    return false;
  }
  
  // 再检查目标词
  return targetKeywords.some(kw => allText.includes(kw.toLowerCase()));
}
```

## 关键词优先级

对于健身类，按相关性排序：

| 优先级 | 关键词类型 | 示例 |
|-------|-----------|------|
| 高 | Facebook页面/群组名 | fitness motivation, gym motivation |
| 高 | 具体动作名 | squat, deadlift, hipthrust |
| 中 | 肌肉部位 | glute, quads, abs |
| 中 | 训练类型 | legday, hiit, cardio |
| 低 | 通用词 | workout, gym, fitness |

优先级高的关键词匹配后，可以更确信这是目标内容。
